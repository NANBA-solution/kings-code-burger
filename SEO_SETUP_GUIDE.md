## 1. Google Search Console Setup

### Step 1: Add Property
1. Go to [Google Search Console](https://search.google.com/search-console)
2. Click "Add Property"
3. Enter your domain: `kingscodeburger.com`

### Step 2: Verify Ownership
Replace the verification code in `client/index.html`:
```html
<meta name="google-site-verification" content="YOUR_VERIFICATION_CODE_HERE" />
```

### Step 3: Submit Sitemap
1. In Search Console, go to "Sitemaps"
2. Submit: `https://kingscodeburger.com/sitemap.xml`
3. Monitor crawl status and indexing

### Step 4: Monitor Performance
- Check "Performance" tab for search queries and click-through rates
- Monitor "Coverage" for any indexing issues
- Review "Enhancements" for structured data validation

## 9. Troubleshooting

### Issue: Not appearing in Google Search
**Solution:**
1. Verify ownership in Search Console
2. Submit sitemap
3. Check robots.txt allows crawling
4. Wait 2-4 weeks for indexing

### Issue: Structured data not showing in Rich Results
**Solution:**
1. Use Rich Results Test to check for errors
2. Verify JSON-LD syntax
3. Ensure all required fields are present
4. Clear Search Console cache

### Issue: Low local search visibility
**Solution:**
1. Complete Google My Business profile
2. Add more reviews
3. Register on local directories
4. Ensure consistent NAP across all platforms
5. Add local keywords to content
